Curso de Codigo Facilito: Git

A tener en cuenta:

git branch: Es para ver las ramas.

git branch namebranch: Es para crear una rama (namebranch) el cual hay va el nombre que se desea.

git merge rama: Es para fusionar ramas.

git checkout rama: Para cambiar de rama.

git branch -d rama: Es para eliminar una rama.

git checkout -b rama: Es para crear una rama y se mueve de una para esa rama.

git clone: Sirve para clonar un rpyecto.
